## 🚀 Getting Started

### Prerequisites

- Node.js (v14 or higher)
- npm or yarn

### Running Projects

#### For Create React App projects:

```bash
cd [project-folder]
npm install
npm start
```

#### For Vite projects:

```bash
cd [project-folder]
npm install
npm run dev
```

#### For projects with backend:

```bash
# Start backend (in backend folder)
cd [project-folder]/backend
npm install
npm start

# Start frontend (in project root)
npm install
npm run dev
```
